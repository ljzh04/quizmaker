# QuizMaker

A simple static quiz generator UI prototype.

## Getting Started

- Open `index.html` in a browser (it redirects to `landing page.html`).
- Use the provided pages to view the quiz flow:
  - `landing page.html`
  - `setup parameter.html` (quiz configuration)
  - `quiz.html` (sample quiz interface)
  - `result.html` (sample results screen)
  - `library.html` (library / quiz list)

## Notes

- This project is a static HTML prototype. There is no build step.
- `setup parameter.html` contains a stubbed-out script that can integrate with an AI backend.
